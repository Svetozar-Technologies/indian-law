# Legal Notes

This project republishes official public law sources with added processing metadata, source links, section anchors, language availability records, and repository-specific editorial notes.

## Source Attribution

Primary source:

- India Code, operated for the Legislative Department, Ministry of Law and Justice: https://www.indiacode.nic.in/

Regional-language source discovery:

- Legislative Department regional-language pages: https://lddashboard.legislative.gov.in/regional-language

Generated pages include source URLs for every law entry and generated Markdown part. When source PDFs are available but section-level text extraction is not yet available, the site records the official file link rather than presenting guessed text.

## Publication Posture

The project is designed around Section 52(1)(q) of the Copyright Act, 1957, including the requirement that Acts be reproduced with commentary or other original matter. The repository therefore adds original metadata and processing notes to each generated law page instead of publishing a bare, unattributed mirror.

For translations, generated pages must keep a prominent note that translations are official source files only when the upstream source identifies them as such. If this project later adds its own translation or OCR correction, that output must be marked as unauthorised unless accepted by the Government of India.

## Disclaimer

This file is an engineering compliance note, not legal advice. Before publishing a full production corpus, obtain review from counsel qualified in Indian copyright and government-publication law.
